## 1. 개요

Round 1은 **강화학습 기반 Time Trial 라운드**로,  
정해진 트랙에서 **가장 빠른 한 바퀴 기록**을 만드는 것이 목표였습니다.

이 라운드에서는 단순히 속도만 높이는 것이 아니라,

- 트랙 이탈(off-track) 최소화
- 주행 안정성 확보
- 속도와 안정성의 균형
- 실차 주행 가능성 검토

를 함께 고려하며 보상함수를 단계적으로 개선했습니다.

## 3. 목표

Round 1의 목표는 다음과 같았습니다.

- 중심선 기반 기본 주행 정책 학습
- 빠른 랩타임 달성
- 오프트랙 최소화
- 실차에서도 재현 가능한 안정적 주행 정책 확보

초기에는 빠른 주행을 우선으로 모델을 설계했지만,  
실험이 진행될수록 **“빠르지만 불안정한 모델”보다 “조금 느려도 안정적인 모델”이 실제 대회와 실차 환경에서 더 유리하다**는 점을 확인했습니다.

---

## 4. 접근 방식

Round 1에서는 하나의 보상함수를 한 번에 완성하기보다,  
여러 모델을 순차적으로 설계하고 실험 결과를 반영하여 점진적으로 개선했습니다.

전체 흐름은 다음과 같습니다.

1. 중심선 + 속도 기반 기본 모델 설계
2. 오프트랙 강패널티를 통한 안정성 보강
3. 레이싱 라인 기반 보상 실험
4. 최종적으로 거리 / 방향 / 속도 스무딩을 포함한 다단계 보상 구조 설계

---

## 5. 모델 발전 과정

### Model 1 - 중심선 + 속도 기반 기본 모델

첫 번째 모델은 가장 기본적인 형태의 보상함수로,  
차량이 중심선에 가깝게 유지되면서 일정 속도를 내도록 유도했습니다.

**핵심 요소**
- `distance_from_center`
- `speed`

**설계 의도**
- 트랙 중앙을 따라 안정적으로 주행
- 너무 느린 주행을 피하고 일정 속도 확보

**한계**
- 속도 확보는 가능했지만 오프트랙이 자주 발생
- 코너 구간에서 안정성이 부족

---

### Model 2 - 오프트랙 억제 중심 모델

두 번째 모델에서는 오프트랙 문제를 줄이기 위해  
`all_wheels_on_track == False` 일 때 매우 작은 보상을 부여하도록 설계했습니다.

또한 지나치게 공격적인 주행을 막기 위해  
속도 설정과 행동 공간을 보수적으로 조정했습니다.

**핵심 요소**
- `all_wheels_on_track`
- 중심선 거리
- 속도 제한

**설계 의도**
- 트랙 이탈을 강하게 억제
- 불안정한 고속 주행 대신 완주 중심 정책 형성

**장점**
- 오프트랙 감소
- 주행 안정성 향상

**한계**
- 속도가 낮아져 랩타임이 희생됨
- 안정성은 좋아졌지만 최고 성능은 부족

---

### Model 3 - 레이싱 라인 기반 모델

세 번째 모델에서는 중심선 기준에서 벗어나  
레이싱 라인을 직접 반영한 보상함수를 실험했습니다.

레이싱 라인 설계를 위해 다음과 같은 요소를 활용했습니다.

- `x`
- `y`
- `speed`
- `timeFromPreviousPoint`

**설계 의도**
- 단순 중앙 유지가 아니라 최적 경로를 학습
- 코너에서 더 효율적인 주행 유도
- 최고 랩타임 개선

**장점**
- 최고 기록 자체는 개선 가능성 확인
- 보다 공격적인 주행 전략 실험 가능

**한계**
- 평균 성능이 불안정
- 수렴 과정이 일정하지 않음
- 빠르지만 재현성이 떨어질 수 있음

---

### Final Model - 3단계 보상 구조

최종 모델은 앞선 실험 결과를 바탕으로  
속도, 안정성, 방향 정합성을 함께 반영하는 구조로 설계했습니다.

#### 1단계: 거리 기반 보상
- 중심선에서 너무 멀어지지 않도록 유도
- 기본 주행 안정성 확보

#### 2단계: 방향 정합성 보상
- 트랙 진행 방향과 차량 heading 차이를 줄이도록 설계
- 코너 진입/탈출 시 불안정성 완화

#### 3단계: 속도 유지 + 스무딩
- 일정 속도를 유지하도록 유도
- 조향 변화가 급격하지 않도록 보상 구조를 조정

**최종 의도**
- 단순히 빠른 모델이 아니라
- 실제 환경에서도 안정적으로 재현 가능한 모델 설계

---

## 6. 보상함수 설계 핵심

Round 1에서 중요하게 본 설계 포인트는 다음과 같습니다.

### 1) 중앙 유지
트랙을 이탈하지 않기 위한 가장 기본 조건으로  
중심선 거리 기반 보상을 사용했습니다.

### 2) 오프트랙 강패널티
빠른 주행을 하더라도 트랙을 벗어나면 의미가 없기 때문에  
오프트랙은 매우 작은 보상으로 처리했습니다.

### 3) 방향 정합성
차량이 트랙 진행 방향과 크게 어긋나면  
다음 구간에서 불안정한 주행으로 이어질 가능성이 높았습니다.

### 4) 속도와 안정성의 균형
무조건 속도를 높이면 기록은 빨라질 수 있지만  
실제 주행에서는 오히려 재현성이 떨어질 수 있어  
속도만이 아니라 **안정적인 코너 통과**도 함께 고려했습니다.

---

## 7. 결과 요약

| Model | 특징 | 장점 | 한계 |
|---|---|---|---|
| Model 1 | 중심선 + 속도 | 기본 주행 가능 | 오프트랙 발생 |
| Model 2 | 오프트랙 강패널티 | 안정성 향상 | 속도 저하 |
| Model 3 | 레이싱 라인 기반 | 최고 기록 개선 가능성 | 수렴 불안정 |
| Final Model | 거리 + heading + speed smoothing | 안정성과 재현성 향상 | 추가 미세 조정 여지 존재 |

Round 1을 통해 확인한 핵심은 다음과 같습니다.

- 속도만 높인다고 좋은 모델이 되는 것은 아니다
- 오프트랙 억제만 강하게 하면 지나치게 느려질 수 있다
- 레이싱 라인 기반 접근은 강력하지만 튜닝 난이도가 높다
- 실제로는 여러 보상 항목의 균형 설계가 가장 중요하다

---

## 8. 실주행 평가

Round 1에서는 시뮬레이션 결과뿐 아니라  
실제 차량 주행 가능성도 함께 확인했습니다.

실차 주행을 통해 다음과 같은 점을 확인했습니다.

- 시뮬레이션에서 좋은 결과가 나와도 실차에서는 다르게 동작할 수 있음
- 차량 상태, 캘리브레이션, 조향 특성 등이 실제 결과에 큰 영향을 줌
- 가장 빠른 모델보다 **안정적으로 반복 주행 가능한 모델**이 더 실전적임

즉, Round 1은 단순한 보상함수 실험이 아니라  
**sim-to-real 차이를 체감하고 설계 방향을 수정한 출발점**이었다고 볼 수 있습니다.

## 11. 배운 점

Round 1을 통해 얻은 가장 큰 교훈은 다음과 같습니다.

- 강화학습 모델의 성능은 보상함수 설계에 크게 좌우된다
- 속도와 안정성은 항상 trade-off 관계를 가진다
- 레이싱 라인 기반 접근은 효과적이지만 정교한 튜닝이 필요하다
- 실제 대회에서는 최고 성능보다 안정적인 재현성이 더 중요할 수 있다
- 이후 Round 2, Round 3에서는 이 경험을 바탕으로 센서 확장과 일반화 성능 확보를 시도했다
