## 1. 개요

Round 3는 대회 당일 공개되는 트랙 또는 사전에 충분히 학습되지 않은 트랙에 대응해야 하는 라운드였습니다.

앞선 Round 1, Round 2에서 얻은 경험을 바탕으로,  
Round 3에서는 두 가지 방향을 모두 고려했습니다.

- 처음 보는 트랙에서도 안정적으로 달릴 수 있는 **범용 모델**
- 특정 트랙에서 최고 기록을 목표로 하는 **최적화 모델**

즉, Round 3의 핵심은  
**“일반화 성능”과 “트랙 특화 성능”을 모두 비교하고 설계하는 것**이었습니다.

---

## 2. 대회 조건

| 항목 | 내용 |
| ---- | ---- |
| Track | 대회 당일 공개 |
| 방향 | 반시계 |
| 알고리즘 | 강화학습 |
| 레이스 | Time Trial |
| 패널티 | 트랙 아웃 +2초 |
| 미션 | 3분 세션 × 2, 세션당 Timeout 1분 |

---

## 3. 목표

Round 3의 목표는 다음과 같았습니다.

- 사전 학습되지 않은 트랙에서도 안정적으로 주행하기
- 특정 트랙에서는 랩타임을 극대화하기
- 일반화 성능과 최적화 성능의 차이를 비교하기
- 실제 주행에서도 재현 가능한 모델 설계하기

Round 1이 주행 안정성 확보,  
Round 2가 장애물 회피와 센서 확장이었다면,  
Round 3는 그 경험을 바탕으로 **“얼마나 범용적인 정책을 만들 수 있는가”**를 검증하는 단계였습니다.

---

## 4. 접근 방식

Round 3에서는 하나의 방향만 고집하지 않고,  
두 개의 전략을 병행했습니다.

### 전략 1: 범용 모델
- 특정 트랙에 과도하게 맞추지 않음
- heading, 속도, completion 중심 보상 설계
- 처음 보는 트랙에서도 안정적으로 주행 가능한 정책 목표

### 전략 2: 특정 트랙 최적화 모델
- 레이싱 라인을 추출해 보상함수에 반영
- 트랙 특성에 맞는 speed/time 정보 활용
- 특정 환경에서 최고 기록 달성 목표

즉, Round 3는 단순히 하나의 최종 모델을 만드는 과정이 아니라,  
**범용성과 최고 성능 사이의 trade-off를 실험적으로 비교한 라운드**였습니다.

---

## 5. 범용 모델

### 설계 방향

범용 모델은 특정 트랙에서의 절대 최고 기록보다는  
처음 보는 트랙에서도 무너지지 않는 안정성을 중요하게 보았습니다.

주요 반영 요소는 다음과 같습니다.

- heading alignment
- speed control
- completion bonus
- off-track penalty

### 설계 의도

- 트랙 진행 방향과 차량 방향이 크게 어긋나지 않도록 유도
- 지나치게 공격적인 조향을 줄이고 안정적 주행 확보
- 완주 비율을 높이고 전체 주행 흐름 유지
- 트랙이 바뀌어도 일정 수준 이상의 성능 유지

### 학습 과정

- 초기에는 기본 학습 파라미터를 적용
- 이후 최저/최고 속도를 조정하면서 추가 학습
- 안정적인 reward 수렴을 확인하며 일반화 성능을 관찰

### 평가

학습에 직접 사용하지 않은 트랙에서도 테스트한 결과,  
off-track 없이 평균적인 주행 성능을 유지할 수 있음을 확인했습니다.

특히 RL Speedway와 같은 이미 익숙한 트랙뿐 아니라,  
새로운 형태의 트랙에서도 완주 가능성이 유지된다는 점이 중요했습니다.

### 의미

범용 모델은 특정 트랙에서 최고 기록을 내는 데에는 한계가 있을 수 있지만,  
대회 당일 공개 트랙이나 예측하기 어려운 환경에서  
**안정적으로 대응할 수 있는 기본 정책**으로서 큰 의미가 있었습니다.

---

## 6. 특정 트랙 최적화 모델

Round 3에서는 범용 모델과 별개로,  
특정 트랙에서 최고 성능을 내기 위한 레이싱 라인 기반 최적화 모델도 설계했습니다.

검토한 주요 트랙은 다음과 같습니다.

- RL Speedway
- A to Z Speedway
- Smile Speedway

이 중 RL Speedway는 이미 기존 라운드에서 충분히 사용한 트랙이기 때문에,  
Round 3에서는 A to Z Speedway와 Smile Speedway 중심으로 최적화를 진행했습니다.

---

## 7. A to Z Speedway

### 1차 모델

A to Z Speedway에서는 레이싱 라인을 추출하고,  
각 포인트별 정보를 기반으로 보상함수를 설계했습니다.

사용한 주요 정보:
- `x`
- `y`
- `speed`
- `timeFromPreviousPoint`

### 설계 의도

- 직선 구간과 코너 구간에서 서로 다른 최적 속도 반영
- 중심선이 아니라 실제 빠른 경로를 따라가도록 유도
- 랩타임 단축에 집중

### 결과

초기 모델은 완주율 자체는 확보했지만,  
오프트랙이 간헐적으로 발생하고 평균 성능의 편차가 존재했습니다.

즉, 빠른 기록 가능성은 있었지만  
아직 완전히 안정적인 상태는 아니었습니다.

### 원인 분석

성능이 기대만큼 나오지 않았던 주요 원인 중 하나는  
Action Space 설정이 트랙 특성에 충분히 맞지 않았기 때문이라고 판단했습니다.

트랙에 비해 조향 및 속도 조합이 최적화되지 않으면,  
레이싱 라인 기반 보상함수를 사용하더라도 실제 주행 품질이 떨어질 수 있었습니다.

### 개선 모델

이후 Action Space를 조정한 뒤 다시 학습을 진행했습니다.

그 결과,
- 안정적인 수렴
- 높은 완주율
- 더 빠른 평균 랩타임
- 실제 주행에서 off-track 감소

를 확인할 수 있었습니다.

### 의미

A to Z Speedway 실험은  
레이싱 라인 기반 보상함수만큼이나  
**Action Space 설계가 중요하다**는 점을 보여준 사례였습니다.

---

## 8. Smile Speedway

Smile Speedway 역시 트랙에 특화된 최적화 모델을 별도로 설계했습니다.

### 설계 방식

A to Z와 유사하게 레이싱 라인을 추출하고,  
다음 정보를 보상함수에 반영했습니다.

- `x`
- `y`
- `speed`
- 예상 주행 시간

### 학습 전략

Smile Speedway에서는 단순히 빠른 경로를 따라가는 것에 그치지 않고,  
목표 랩타임 기반의 추가 보상을 도입했습니다.

#### 1차 학습
- 레이싱 라인 기반 보상 설계
- 기본적인 빠른 경로 추종 학습

#### 2차 학습
- 목표 랩타임 도달 시 추가 보상 부여
- 평균 랩타임 개선을 유도하는 구조 적용

### 결과

Smile Speedway 모델은
- 안정적인 완주
- off-track 없는 주행
- 평균 랩타임 개선

측면에서 좋은 결과를 보였습니다.

### 의미

Smile Speedway 실험은  
레이싱 라인 자체뿐 아니라  
**목표 시간 기반 보상 설계가 성능 개선에 도움이 될 수 있다**는 점을 보여주었습니다.

---

## 9. 보상함수 설계 핵심

Round 3에서 중요하게 본 설계 포인트는 다음과 같습니다.

### 1) 일반화 vs 최적화의 구분
모든 트랙에서 무난하게 주행하는 정책과  
하나의 트랙에서 최고 성능을 내는 정책은 설계 철학이 다릅니다.

### 2) heading alignment의 중요성
새로운 트랙에서는 중심선보다  
현재 진행 방향과 차량 heading의 정렬이 더 안정성에 직접적인 영향을 주는 경우가 많았습니다.

### 3) completion 중심 보상
처음 보는 트랙에서는 최고 기록보다  
우선 완주 가능한 정책을 형성하는 것이 더 중요했습니다.

### 4) 레이싱 라인 정보의 활용
특정 트랙 최적화에서는
레이싱 라인, 구간별 적정 속도, 예상 소요 시간을 함께 고려하는 것이 효과적이었습니다.

### 5) Action Space의 영향
보상함수만 잘 만든다고 충분하지 않았고,  
실제로는 Action Space 설정이 모델 성능에 매우 큰 영향을 주었습니다.

## 10. 결과 요약

| 모델 | 목표 | 특징 | 결과 |
|---|---|---|---|
| 범용 모델 | 일반화 | heading / speed / completion 중심 | 새로운 트랙에서도 안정 주행 |
| A to Z 1차 | 트랙 최적화 | 레이싱 라인 기반 | 빠르지만 불안정성 존재 |
| A to Z 개선 | 트랙 최적화 | Action Space 조정 후 재학습 | 성능과 안정성 동시 개선 |
| Smile Speedway | 트랙 최적화 | 목표 랩타임 기반 추가 보상 | 안정적이며 빠른 주행 |

Round 3를 통해 확인한 핵심은 다음과 같습니다.

- 일반화 성능과 최고 기록은 서로 다른 설계를 요구한다
- 새로운 트랙에서는 안정성 중심 설계가 우선이다
- 특정 트랙에서는 레이싱 라인 기반 최적화가 매우 효과적이다
- 보상함수와 Action Space는 함께 설계해야 한다

## 11. 실주행 평가

Round 3에서는 시뮬레이션 결과뿐 아니라  
실제 주행에서 재현 가능한지도 함께 확인했습니다.

실험을 통해 다음과 같은 점을 확인했습니다.

- 범용 모델은 실제 환경에서도 비교적 안정적이었다
- 특정 트랙 최적화 모델은 매우 빠를 수 있지만 세팅 민감도가 높았다
- 실차에서는 작은 조향 편차나 환경 변화도 결과에 영향을 줄 수 있었다
- 결국 대회 환경에서는 최고 기록뿐 아니라 재현성이 매우 중요했다

즉, Round 3는  
**“가장 빠른 모델”과 “가장 실전적인 모델”이 다를 수 있음을 명확히 보여준 라운드**였습니다.

## 14. 배운 점

Round 3를 통해 얻은 가장 큰 교훈은 다음과 같습니다.

- 일반화 성능과 최고 성능은 서로 다른 목표다
- 새로운 트랙에 대응하려면 안정성 중심 설계가 중요하다
- 특정 트랙 최적화에서는 레이싱 라인과 Action Space가 핵심이다
- 실제 대회에서는 빠른 한 번보다 안정적으로 반복 가능한 주행이 더 중요할 수 있다
- 전체 프로젝트를 통해 보상함수 설계, 센서 활용, Action Space, 실차 검증이 모두 함께 고려되어야 한다는 점을 확인했다
